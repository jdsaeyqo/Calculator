# 간단 계산기

- 기본적인 계산기능을 제공한다.  
- 연산자는 한 번만 사용할 수 있다.  
- 계산 기록들을 로컬에 저장하여 보여준다.

# 사용 기능

 - ConstraintLayout, TableLayout, LayoutInflater, Thread, Room
